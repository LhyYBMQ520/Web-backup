# L菌の导航站
个人主页 & 导航站点开源备份仓库，**纯前端静态页面**，无后端依赖，推荐使用 Nginx 部署。

## 🌐 在线访问
双栈官方访问地址：
> https://www.lcatnya.top/

## 🚀 快速部署
1. 克隆本仓库到本地
```bash
git clone https://github.com/LhyYBMQ520/Web-backup
```
2. 将仓库内静态文件放入 Nginx 站点根目录，主入口文件为默认的index.html
3. 配置 Nginx 并重启，即可直接访问使用
